f: focal length; u: distance of the object; v: distance of the image.

Use the combo box to choose which variable you want to solve for.

Move the slider or click the spinner to set the values of f, u or v.

You can also input the value directly.

Please note that the range of f, u and v is restricted.

The spinner on the bottom right is used to set the size of the object.

Click "start tutorial" to start pre-set tutorial.
